Cryptology


Cryptography : art of making ciphers -> it is further divided into 2 types classical & modern.
Cryptanalyst : art of breaking ciphers -> it's further divided into 2 types classical & modern.
                        classical cryptanalysis --> mostly done by hands or early computing techniques.
                        modern cryptanalysis --> it's done by modern computing techniques.

Cryptography - Basic Terms:
  • Cipher :  Encryption method or algorithm
  • PlaintText : Non-encrypted text
  • CipherText : Encrypted text
  • Key : Secret information used for encryption & decryption.
  • Alphabet (plaintext or ciphertext alphabet) :  (Ordered) set of letters or symbols which are used for encryption/decryption.

Note : If u are creating a cipher make sure your cipher should be secure by an attacker if he knows the cipher & how it works but not having the key.
    
Types Of Classical Ciphers: It has 3 types
  1. Substitution Ciphers 
    • Replace letters by other letters or symbols
    • Eg: Ceaser Cipher , Simple MASC , Vigenere
  2. Transposition Ciphers
    • change the order of plaintext letters
    • Eg: Scytale , columnar transposition
  3. Composed Ciphers
    • Combination of substitution & transposition cipher
    • Eg: ADFGVX , Granite


Cryptography - Other Basic Terms:

  • Keyspace size : cardinatlity of keyspace set & usually cryptographers rounded up the value in power of 2

Basic Terminologies For Substitution Cipher:
  • Monoalphabetic Substitution
    • Only 1 cipher text alphabet is used
    • Examples : Ceaser cipher , simple MASC
  • Polyalphabetic Substitution
    • The cipher text alphabet changed during encryption
    • Examples : Vigenere cipher , Enigma machine
  • Homophonic Substitution
    • A letter is encrypted by morethan 1 letter or symbol
    • Eg: Zodiac Killer ciphers , historic ciphers of vatican
  • Polyphonic Substitution
    • Different plaintext letters are encrypted by the same cipher text letters.
    • Non-deterministic. Decryption ambiguous
    • Eg: historic ciphers of the vatican


Cryptography - Other Basic Terms:
  • Monographic Cipher
    • One letter is encrypted at the same time (Ceaser Cipher)
  • Bigraphic Cipher
    • Letters pair are encrypted at the same time (Playfair Cipher)
  • Monopartite Cipher
    • Replacement is a single letter (Ceaser Cipher)
  • Bipartitie Cipher 
    • Replacement are two letters (ADFGVX)

            NOTE : In the above paragraphs graphic refers plaintext & partite refers ciphertext
            Example : The simple MASC is a monoalphabetic monographic monopartite substitution cipher

Cryptanalysis - Basic Terms:
  • Cryptanalyst
    • Someone who analyzes a cipher or ciphertext to break it
  • Attack
    • Method to revert key or plaintext of a ciphertext
  • Breaking  a ciphertext
    • Sucessfully performed attack on a single ciphertext
  • Breaking  a cipher
    • Finding an attack on a cipher that works reproducibly on ciphertexts encrypted with the particular cipher 




Cryptanalysis - Attack Types:
  • Assumption with each attack type
    • "Attacker knows the system" (i.e the used cipher , no security through obscurity)            
  • Chosen plaintext attack
    • Goal: revert the key
    • Attacker is able to produce (arbitary) plaintext-ciphertext pairs.
  • (Partially) known plaintext attack
    • Goal: revert the key , revert the rest of unknown plaintext
    • Attacker has (part of) the plaintext of  a ciphertext
  • Ciphertext-only attack
    • Goal: revert the key , revert the plaintext
    • Attacker only is in the possesssion of the ciphertetxt




Cryptanalysis - Attacking Ciphers
  • Bruteforce Attack (aka exhaustive key search)
    • Attack that works with every cipher ( except perfect ciphers, e.g the one time pad)
    • Attacker tests every key of the cipher
    • Only suitable, if it's practical to search through the keyspace,
  • Manual Attack
    • E.g. break a MASC by hand using the knowledge of letter frequency distribution
    • E.g. cut transposition ciphertext into paper strips & rearrange them
  • Computerized Attack
    • Implementation of manual attacks, e,g, automated frequency analysis
    • Heuristic attacks work on many classical ciphers, e.g. MASC , transposition ,
